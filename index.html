Arcana

Arcana is a Tarot card game composed of the Major Arcana of a Tarot deck. It is a clickable three-card spread that randomly displays a card and its corresponding description for one's past, present, and future. It was built using JavaScript and jQuery to create an array of cards and a function to randomly deal a card and its corresponding description. I also added a chance for the card to display upside down which would give a negative reading.
