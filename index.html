The Rubbing Lamp

The Rubbing Lamp, is a single page crowdfunding platform inspired by Kickstarter. The app allows the user to create and edit projects filtered by their category. It utilizes AWS S3 and Paperclip for image uploading and storage. I implemented the smooth user interface with React.js and used Rails with a PostgreSQL database for the back end.

Hamilton Inn

Hamilton Inn is a website for guests to make inquiries to stay in a five-story townhouse in Manhattan, New York. Using Bootstrap and SCSS, I responsively built and designed haminn.com. The smooth interface of the svg of the house was built with D3.js. Using Google Forms, visitors can make inquiries based on their travel plans. These inquiries are sent via email and through slack to the administrators.
